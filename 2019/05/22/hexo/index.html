下载 node.js
使用 国内npm镜像

npm config set registry https://registry.npm.taobao.org
验证是否成功

npm config get registry
安装hexo

npm install hexo-cli -g
检查版本

hexo -v
初始化文件夹

hexo init
安装所需组件

nmp install
在本地生成静态文件

hexo generate
启动本地服务

hexo sever
修改 -config.yml文件

deploy：

安装 部署到git的工具

npm install hexo-deployer-git –save

hexo clean 清空
hexo generate
hexo deploy 推上去

新建文件 CNAME

hexo s 本地生成地址先浏览一下
hexo g
hexo d
